# Sidebar Toggle — keyboard shortcuts

| Shortcut | Action | Works out of the box? |
|----------|--------|------------------------|
| **Ctrl+Shift+E** | Toggle hide on the current supported site (Wikipedia · old Reddit · YouTube · GitHub) | Chrome: **yes** · Edge: assign manually \* |
| **Alt+E** | YouTube only — toggle cinema dim (spotlight the player) | Assign manually in Chrome & Edge \* |

\* **Why "assign manually"?** Browsers reserve some key combos for themselves — **Edge** uses `Ctrl+Shift+E` for sidebar search, and **both Chrome and Edge** use `Alt+E` to open the browser menu. A reserved combo can't be claimed automatically, so the command shows as **Not set**. To enable it:

1. Open `chrome://extensions/shortcuts` (or `edge://extensions/shortcuts`).
2. Find **Sidebar Toggle**.
3. Click the field next to the action and press the keys.

Once assigned, the extension's binding takes priority over the browser's. Every action is also available from the toolbar popup, so the keyboard shortcuts are entirely optional.
